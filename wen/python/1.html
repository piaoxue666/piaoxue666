飘血

单行注释：

# 注释内容

多行注释：

'''中间可以写很多行'''
或
"""中间也可以写很多行"""
误区提示：不可以只写一对引号！

编码声明注释：

语法格式如下
#-*-coding:（编码）-*-
或者
#coding=（编码）
温馨提示：编码一律用小写，'-*-'没有什么特殊作用，只是美观

代码缩进：

缩进可以用'TAB'键或者是'空格键'实现，通常缩进4个格，其他的个数也可以
if a:
    print('666')
误区提示：python对代码的缩进要求非常严格，同一个级别的代码缩进量必须相同，否则，会抛出SyntaxError异常
if a:
    print('666')
  print('555')
这就是错误的

编写规则：

python采用了PEP 8作为编码规范，PEP是Python Enhancement Proposal的缩写，翻译过来就是python增强建议书
1.每个import语句只导入一个模块，尽量避免一次性导入多个模块
2.不要在行尾添加分号';'，也不要用分号将两个命令放在一行
3.建议每行不超过80个字符，如果超过，建议用'()'将它们联系在一起，不推荐用'\'进行链接
4.使用空格可以增加代码的可读性，便于维护
5.通常情况下，运算符两侧、函数参数之间、逗号'，'两侧建议使用空格分隔

保留字：

保留字可以用两行代码查看：
import keyword
print(keyword.kwlist)
输出的结果为：
['False', 'None', 'True', 'and', 'as', 'assert', 'async', 'await', 'break', 'class', 'continue', 'def', 
'del', 'elif', 'else', 'except', 'finally', 'for', 'from', 'global', 'if', 'import', 'in', 'is', 'lambda', 
'nonlocal', 'not', 'or', 'pass', 'raise', 'return', 'try', 'while', 'with', 'yield']
